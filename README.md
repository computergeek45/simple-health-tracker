The Health Tracker Dashboard is a Python Streamlit application that enables users to log daily wellness metrics such as mood, sleep, hydration, exercise, and energy levels.
It computes averages, wellness scores, and visualizes health trends through charts and summaries. Based on tracked data, the app provides rule-based personalized health recommendations. 
Designed with a highly customized modern UI using advanced CSS animations and gradients, this project highlights skills in Python data handling, frontend customization in Streamlit,
and health-focused analytics.

Tech Stack: Python, Streamlit, Pandas, CSS
to run:
 Streamlit filename.py
